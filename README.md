# Week 1 Retail Sales Excel Analysis

## Project Overview

This project was completed as part of the **LogicStack Data Analysis Internship (June 2026)**.

The objective of this project was to analyze a retail sales dataset using Microsoft Excel/WPS Spreadsheet. The project includes data cleaning, data quality checks, basic analysis using Excel formulas, sorting, filtering, charts, and observations.

## Charts Created

* Bar Chart: Total Sales by Product Category
* Column Chart: Total Quantity Sold by Product Category
* Pie Chart: Total Sales by Gender

## Key Observations

* Electronics recorded the highest total sales.
* Beauty recorded the lowest total sales.
* A total of 1000 transactions were analyzed.
* The total quantity sold was 2514 units.
* The highest transaction amount was 2000.
